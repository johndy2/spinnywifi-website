How to Connect

Crucial Steps:

If you want to connect, just make sure to set this as your DNS. We only support DS games for now, and we support authentic hardware and emulators, eg, MelonDS.

Put your primary DNS AS (this is crucial): 71.38.198.179
Leave the secondary DNS as 0.0.0.0 It works for only DS games, Wii support is coming soon if the server works, right now if you intend to connect when the server is not on, you will get 52200 or 52100.
